# Carpet Tile

## What it is
Modular textile flooring in 24×24, 18×36, and plank formats. Unlike broadloom, every tile is a discrete unit: wear hides between modules, individual tiles lift out for replacement, the install pattern itself becomes a design tool, and you only buy as much as you actually need (plus attic stock). For commercial Ottawa work — corporate, institutional, government, education — carpet tile is the default specification in 2026. Broadloom is now the exception, not the rule.

A carpet tile is built in four layers:
1. **Face fibre** — usually Type 6 or Type 6,6 solution-dyed nylon (Antron, Econyl, EcoSolution Q100). Solution-dyed means colour is in the polymer itself — bleach-cleanable, UV-stable, no dye-lot drift.
2. **Primary backing** — woven or non-woven scrim that anchors tufts.
3. **Secondary / dimensional backing** — the layer that gives the tile its shape and stability. This is where the sustainability story lives: Interface GlasBac / CQuestBio / CQuestBioX, Shaw EcoWorx / EcoWorx BIO, Mohawk EcoFlex, Tarkett ethos Modular with Omnicoat.
4. **Tile geometry** — cut to tight tolerances so seams disappear in any pattern.

## Types / sub-products

| Format | Dimensions | Notes | When spec'd |
|---|---|---|---|
| **24×24" square** | 610×610 mm | Legacy standard. Monolithic / quarter-turn / ashlar patterns. Highest availability across every mill. | Default for offices, classrooms, corridors. |
| **50×50 cm square** | 500×500 mm | European-standard metric equivalent — Interface's global default. Nearly identical to 24×24 in-field. | Interface-led specs, LEED projects, anywhere a metric drawing set is in use. |
| **18×36" rectangular** | 457×914 mm | Mid-format rectangle — runs longer pattern fields with fewer seams. Strong ashlar / brick effect. | Corridors, long open-plan floors, wayfinding runs. |
| **9×36" plank** | 229×914 mm (approx.) | Plank format — enables herringbone, chevron, and wood-plank-mimic layouts. | Design-forward spaces, lobbies, hospitality-commercial crossover. |
| **Large-format 36×36"+** | 914×914 mm and up | Fewer seams, faster install, less waste on open floors. Often loose-lay friendly. | Large open office floorplates, warehouse-to-office conversions. |

**Most-spec'd in 2026 commercial (Ottawa market):** 24×24 square remains the workhorse (roughly 55-65% of YCI carpet-tile volume, unverified), but 18×36 and plank are climbing fast in design-led specs. Interface 50×50 cm shows up on every LEED v4.1 / v5 project we've bid in the last 18 months.

## Install techniques

| Technique | Substrate | Best for | Notes |
|---|---|---|---|
| **Full-spread glue-down** | Concrete or plywood | Ramps, heavy rolling loads, wet-adjacent zones, wheeled-traffic corridors | Most stable. Hardest to lift — tile replacement requires cutting. Use when the floor will never be reconfigured. |
| **Tackified release adhesive** | Concrete or plywood | Offices, classrooms — the YCI default | Pressure-sensitive adhesive stays tacky indefinitely. Tiles lift cleanly for replacement or access. This is what we install 80%+ of the time. |
| **Peel-and-stick / self-adhered** | Concrete or plywood (smooth) | Occupied buildings, after-hours work, schools during spring break | Factory-applied adhesive on the tile back. No wet adhesive = no cure time = room back in service same day. Interface TacTiles falls adjacent to this category (see below). |
| **TacTiles (Interface-specific)** | Concrete, plywood, access floor | LEED / low-VOC specs, occupied buildings, access floors | Glue-free 3" adhesive squares that connect tile-to-tile, not tile-to-substrate. Creates a floating floor. 4-5 TacTiles per m². Near-zero VOC. Tiles + TacTiles recycle together through ReEntry. |
| **Loose-lay (large-format only)** | Limited — flat, clean, dry concrete | Temporary installs, quick-change retail, raised access floor | Relies on tile mass + edge friction + perimeter glue. Not suitable for rolling loads or sloped surfaces. |

**Substrate rule of thumb (Ottawa):** On-grade and below-grade concrete slabs in Ottawa basements see enough seasonal moisture swing that we default to release adhesive over peel-and-stick — the adhesive re-engages if the tile lifts slightly during a humid August.

## Install patterns

The install pattern is a design decision with real cost, wear, and replaceability implications. Every carpet tile ships with an arrow printed on the back indicating the tuft direction. The arrow is the reference for every pattern below.

| Pattern | Arrow orientation | Visual read | Best use | Trade-offs |
|---|---|---|---|---|
| **Monolithic** | All arrows same direction | Reads like broadloom — one continuous field | When the client wants "carpet without seams." Corporate boardrooms, executive floors, feature spaces. | Shows every directional wear pattern and seam mis-alignment. Replacement tiles from a different dye lot will glow. Most demanding pattern to install well. |
| **Quarter-turn (ashlar-square / tessellated)** | Each tile rotated 90° from its neighbour, checkerboard | Subtle basket-weave — soft directional cues in both axes | **The YCI default.** Hides traffic wear, hides dye-lot drift, hides installer error, tolerates future tile replacement. Open-plan offices, classrooms, corridors. | None, really. This is why it's the default. |
| **Brick / running bond** | All arrows same direction, rows offset by half a tile | Horizontal brick courses | Long narrow corridors — the offset guides the eye down the run. Lobbies where you want a subtle directional pull toward reception. | Still shows wear like monolithic. The offset helps but doesn't eliminate it. |
| **Ashlar** | All arrows same direction, columns offset by half a tile (perpendicular to brick) | Vertical brick — stacked, more formal | Formal spaces, libraries, conference rooms where you want rhythm without movement. | Same wear visibility as brick. |
| **Herringbone** | Plank tiles laid at 45° (or 90° to each other) in interlocking V's | Strong directional pattern, high-design statement | Lobbies, hospitality, boutique retail, executive reception. **Plank format only** (9×36 or similar). | Cut waste is significant (~10-15% overage vs. 5-8% for square). Install labour 30-50% higher. Mis-cut at perimeter is very visible. |
| **Chevron** | Planks cut on angle, forming continuous V's with no break | Sharper, more uniform than herringbone | Ultra high-end design spaces | Cut waste even higher. Premium labour. |
| **Random / non-directional (ashlar-random)** | Every tile rotated freely, no system | No pattern at all — visual noise hides everything | High-traffic, damage-prone, replacement-heavy floors. Schools, daycares, airports, municipal buildings. | The safest pattern for long-term replaceability. Any new tile from any dye lot blends in. Some designers hate it. |

**The YCI rule:** If the client hasn't specified a pattern and the tile is a plain / tonal style, we install quarter-turn. If it's a plank or a patterned tile, we follow the manufacturer's recommended layout on the product data sheet — every mill publishes one and deviating voids warranty claims on pattern-match defects.

## Best used for

- **Open-plan offices and workstation pods** — replace tiles under failed chair casters or coffee spills without touching the rest of the floor.
- **Schools, colleges, universities** — damage replacement is routine. K-12 corridors in Ottawa-Carleton and OCDSB buildings are almost exclusively carpet tile in 2026.
- **Federal and provincial government offices** — security cable trenches, frequent reconfigurations, raised access floors. Canadian federal specs (PSPC / CCOHS) frequently name NSF/ANSI 140 Gold and CRI Green Label Plus.
- **Healthcare administrative and non-clinical zones** — waiting rooms, admin offices, nurses' stations (not clinical / wet zones).
- **Libraries, community centres, municipal buildings** — City of Ottawa facilities use carpet tile widely in non-wet public spaces.
- **Raised access floor systems** — paired with the right adhesive (usually TacTiles or release-tack), tiles lift with the access panel.
- **Any space where future furniture reconfiguration is expected** — modular systems furniture + modular flooring is the right pairing.

## Best NOT used for

- Wet zones — washrooms, kitchens, locker rooms (use sheet vinyl or sheet rubber)
- Clinical healthcare — ORs, patient rooms, labs (use sheet vinyl with heat-welded seams)
- Food prep / commercial kitchens (use quarry tile or sheet vinyl)
- Outdoor / covered exterior (use specialty matting or sealed concrete)
- Industrial / manufacturing floors with chemical exposure (use epoxy or sheet rubber)

## Interface — deeper dive (flagship brand)

Interface is the origin story of modular carpet — Ray Anderson's mission-zero pivot in 1994 remade how the whole industry thinks about carbon. For 2026 Ottawa commercial work, Interface is the brand we lead with on any sustainability-forward spec.

### Backing systems (the sustainability story)
- **GlasBac** — Interface's legacy standard backing. 50+ years of proven dimensional stability. "Installs flat, stays flat, stays put." Made with recycled content including post-consumer carpet tile from the ReEntry program.
- **CQuestBio** — bio-based and recycled-content backing. Carbon-negative at the backing layer (emits less carbon in manufacture than the bio-materials sequestered).
- **CQuestBioX** — Interface's most carbon-negative backing. Stores more carbon than it emits across the product lifecycle. This is the backing to specify on net-zero-target projects.

Every Interface tile EPD published since 2020 shows cradle-to-gate carbon data in kg CO₂e per m². Ask the rep for the current EPD PDF before spec'ing — the numbers update annually.

### TacTiles installation system
Interface's signature glue-free install. Small adhesive squares (or sheets of 6) connect tile to tile — the floor "floats" on the substrate without any wet adhesive. Key properties:
- 4-5 TacTiles per m² of floor (manufacturer spec)
- Near-zero VOC — suitable for occupied-building work
- Tiles + TacTiles recycle together through ReEntry
- Individual tiles lift and replace without disturbing neighbours
- Rated for commercial traffic, including rolling office chairs
- Formats: CQuestGB & GlasBac connectors in rolls of 500 or sheets of 6

### Flagship tile collections 2026 (verify current availability with rep before spec)
- **Human Nature** — biophilic textures, plank and square formats
- **World Woven** — textile-inspired heritage patterns
- **Composure** — plain tonal workhorse, widest colour range
- **Net Effect** — ocean-theme, incorporates recovered fishing nets (Econyl)
- **Equal Measure** — geometric grid, strong for wayfinding
- **Open Air** — lightweight, low-pile for open-plan

### Interface ReEntry (end-of-life)
Interface's take-back program, launched 1995. In Canada, Interface works with Aspera Recycling Inc. to reprocess end-of-life tiles into reusable fibre and PVC backing for new InterfaceFLOR products. For an Ottawa contractor:
1. Request ReEntry service via the Interface Canada sustainability page.
2. Interface arranges reclaim logistics (palletizing, freight pickup).
3. Tiles go to Aspera (Canadian processor) or, for specific streams, back to a US Interface facility.
4. Tiles are separated into face fibre (recycled into new yarn) and backing (ground and reused).
5. What can't be recycled is converted to energy.

Interface's corporate target is **carbon-negative as a company by 2040**. ReEntry is a core lever in that commitment.

**Logistics note for YCI:** Minimum quantities and freight thresholds apply. For small tear-out jobs (under a pallet), we typically accumulate reclaimed tile at the yard and coordinate a single pickup quarterly. Confirm with the Interface rep — thresholds change.

## Substrate prep + moisture

Carpet tile is tolerant, but not infinitely tolerant. The failure mode when prep is skipped is always the same: telegraphed substrate defects, edge curl, adhesive failure, or mould under the tile.

### Moisture testing (mandatory on concrete)

| Standard | What it measures | When to use | Typical threshold |
|---|---|---|---|
| **ASTM F1869** | Surface moisture vapour emission rate — calcium chloride dish test, 72-hour exposure | Quick pre-install check, smaller jobs | Most mills spec ≤ 3 lbs MVER / 1000 sqft / 24 hr (unverified per mill — confirm) |
| **ASTM F2170** | In-situ relative humidity via probes drilled into the slab | Larger jobs, critical projects, new slabs. Now the preferred method per RFCI. | ≤ 75-85% RH depending on mill and backing — Interface, Shaw, Mohawk publish specific thresholds in their installation guides |
| **ASTM F710** | Concrete pH at the surface | Every install | pH 7-10 typical allowable range |

**YCI standard practice:** On any new concrete or below-grade slab, run both F1869 (surface) and F2170 (internal RH) before scheduling install. F2170 is what the manufacturers actually honour on warranty claims in 2026. Document both results with photos, date, and slab location.

### Flatness and surface prep
- Flatness ≤ 3/16" in 10' (industry standard for carpet tile)
- Clean, dry, sound, smooth, free of dust, oil, paint, curing compounds, sealers, and old adhesive residue
- Embossing leveller required over old VCT if tile is left in place (skim-coat typically 1/16" to 1/8")
- Old adhesive residue: mechanically remove or encapsulate per mill instructions — solvent-based removers usually void the new tile warranty

### Ottawa-specific substrate notes
- Below-grade concrete in Ottawa office towers frequently shows F2170 readings in the 80-90% RH range in spring. Always test — don't assume.
- 1960s-80s office buildings often have original mastic / cutback adhesive under VCT. Encapsulate; don't try to strip.
- Raised access floor (RAF) in federal buildings: use TacTiles or release-tack, never full-spread.

## Maintenance

| Frequency | Task | Notes |
|---|---|---|
| Daily | Vacuum high-traffic lanes | Commercial upright with strong airflow — beater bar OK on loop pile, gentler on cut pile |
| Weekly | Full-floor vacuum | |
| Monthly | Spot clean + interim encapsulation on entry zones | Encapsulation chemistry crystallizes soil for next-vacuum removal |
| 6-12 months | Encapsulation clean floor-wide | Depends on traffic. Schools and government lobbies: 6 months. Executive office: 12+. |
| 18-36 months | Hot water extraction / deep clean | Only on extraction-rated tile backings — verify before scheduling. Some PVC-free tiles cannot be HWE'd. |
| As needed | Individual tile replacement | Lift damaged tile, install replacement from attic stock. Match dye lot when possible. |

**Attic stock rule:** Always spec 5-10% attic stock at install. Dye lots drift. Tile that's been on the floor 3 years has subtly UV-shifted; a brand-new replacement will read slightly different even from the same SKU.
