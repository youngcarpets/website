# Website Improvements — Merged Best-Of from Dev Versions

Curated from V1–V6 dev variants. Reviewed and narrowed down 2026-04-09.

---

## Implement This Round

### Visual Tweaks (Preview & Apply)
1. **Navbar corners** — more squared radius (less pill-shaped)
2. **Map size** — bigger on mobile and desktop
3. **Line animation** — reusable progress/timeline pattern (from V4 "How We Work")
4. **Scroll progress indicator** — line at top showing page position

### Contact Section
5. **V6 contact layout** — names + email + phone only, no label/title/group heading

### Content & Language
6. **No "subcontract" language** — can say "our installers", "our crews" — NEVER use "subcontract" or "subcontractor" on public pages
7. **Content harvesting** — unify best copy from all dev versions per section during implementation

### Product Navigation (Bigger Feature)
8. **"Search by:" section** — purpose, supplier, style — placed ABOVE product tiles, accordion/collapsed UI
9. **Remove "also" list** — everything lives in type tiles or search-by tree
10. **Product categories** — reorganize into: Soft Surfaces, Hard Surfaces, Resilient, Rubber & Impact, Wood + Natural, Specialty Flooring, Accessories (transitions, glues, caps, nosings, treads, risers, tools, etc.)

### Design Rules
11. **Greyscale palette** — charcoal/grey/black/white/greige. Color ONLY for accents (underlines, glows, outlines, pills). Headings/logo color comes at the very end. Strip imported features to greyscale first.
